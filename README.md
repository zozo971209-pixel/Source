# 源 - 哲學探索之旅

這是一個關於哲學探索的網站，涵蓋了從古希臘到當代的智慧傳承。

## 網站功能
- 探索不同時期的哲學思想
- 了解著名哲學家的生平與理論
- 響應式設計，支援多種裝置

## 如何部署
1. 前往 GitHub 倉庫的 **Settings**。
2. 點擊左側選單的 **Pages**。
3. 在 **Build and deployment** 下，選擇 **Deploy from a branch**。
4. 選擇 `main` 分支並點擊 **Save**。
5. 等待幾分鐘後，您的網站將會上線。

## 檔案結構
- `index.html`: 網站主入口
- `assets/`: 包含編譯後的 JS 與 CSS 檔案
- `philosophers/`: 哲學家相關圖片
- `avatars/`: 頭像圖片
- `eras/`: 不同時期的背景圖片
